# Part A: Multiplication Table Generator

## Objective

Create a program that accepts a number from the user and displays its multiplication table up to 10.

## Concepts Used

* User Input
* Variables
* for Loop
* Arithmetic Operations

## Sample Output

```
Enter a number: 7

7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
7 x 10 = 70
```

# 🚀 How to Run

## Python Programs

Open terminal and run:

```bash
python filename.py
```

Example:

```bash
python multiplication_table.py
```

---

## C Programs

Compile:

```bash
gcc filename.c -o output
```

Run:

```bash
./output
```

Example:

```bash
gcc multiplication_table.c -o table
./table
```
