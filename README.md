# Part A: Multiplication Table Generator

## Objective

Create a program that accepts a number from the user and displays its multiplication table up to 10.

## Concepts Used

* User Input
* Variables
* for Loop
* Arithmetic Operations

## Sample Output

```text
=== Multiplication Table Generator ===

Enter a number: 7

7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
7 x 4 = 28
7 x 5 = 35
7 x 6 = 42
7 x 7 = 49
7 x 8 = 56
7 x 9 = 63
7 x 10 = 70
```

# Part D: Password Attempt Simulator

## Objective

Create a simple authentication system that:

* Stores a predefined password
* Allows only 3 attempts
* Grants access for correct password
* Locks account after three failed attempts

## Concepts Used

* Loops
* Conditions
* String Comparison
* Authentication Logic

## Sample Output

```text
=== Password Attempt Simulator ===

Attempt 1: Enter Password: abc
Incorrect Password
Remaining Attempts: 2

Attempt 2: Enter Password: xyz
Incorrect Password
Remaining Attempts: 1

Attempt 3: Enter Password: admin123

Access Granted
```
